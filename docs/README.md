# 介绍

ReportDesign是一款基于ElementUI二次开发的Vue组件库

核心理念是：以数据驱动试图

只需要100行代码就可以生成一个带有分页查询、条件查询和排序功能的完整report页面，并可以响应操作按钮事件

以配置的形式去生成页面，只需很少的html代码，使用者不用关心样式问题，只需要从业务的角度去实现功能就行，减少重复造轮子的工作

比较适用于管理后台系统和低代码平台
